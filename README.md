# Power-BI

## Day1
**topics coverd**
1. Bar Chart – compares values between categories
2. Column Chart – compares values vertically
3. Line Chart – shows change over time
4. Pie Chart – shows percentage or share
5. Donut Chart – similar to pie chart
6. Table – shows data in rows and columns
7. Card – shows one important number (total, count)

## Day2 ##
**topics coverd**
1. Hierarchies in Matrix – shows data in levels (Year → Month → Day)
2. Subtotals in Matrix – shows total for each group
3. Matrix – displays data in rows and columns with group
4. Number Formatting in Matrix – formats numbers (currency, percentage, decimals)

## Day3 ##
**Topics Covered**
1. Simple Line Chart to show trends over time.
2. Drill Down Chart to analyze data from high level to detailed level.
3. Scatter Chart to compare two numeric values and find relationships.
4. Waterfall Chart to understand step-by-step increase and decrease.
5. These charts help in clear data visualization and analysis.
